Primeiras formatações

Negrito / destaque

Tu pode usar a tag B para escrever em negrito, porem ela nao eh semantica

ou tu usa a tag STRONG para escrever em negrito, que ja eh semantica

Italico / enfase

tu pode usar a tag I para escrever em Italico, porem nao eh semantica

ou tu usa a tag EM para escrever em negrito, essa tag ja eh semantica

Texto marcado

Tu pode criar um texto marcado com a tag MARK

Texto grande e peqeno

com a tag big a letra fica maior(esta tag esta absoleta)

com a tag small a letra fica menor

Texto deletado

Vc pode fazer q um texto eh deletado com a tag DEl

Texto inserido

Ou tu insere um texto com a tag INS para dar uma enfase e indicar q ele foi adc depois

E para sublinhar use a tag U, ela nao eh semantica

Texto subrescrito

Para colcoca um numero elevado usa a tag SUP, exemplo = 202 = 200

Texto subscrito
